CMAKE_DISABLE_PRECOMPILE_HEADERS¶

Added in version 3.16.

Default value for DISABLE_PRECOMPILE_HEADERS of targets.

By default CMAKE_DISABLE_PRECOMPILE_HEADERS is OFF.
