cmake-find-required¶

  • The CMAKE_FIND_REQUIRED variable was added to tell find_package(), find_path(), find_file(), find_library(), and find_program() to be REQUIRED by default. The commands also gained an OPTIONAL keyword to ignore the variable for a specific call.
