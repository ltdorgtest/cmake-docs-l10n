CPACK_PERMANENT¶

Added in version 3.1.

Request that this file not be removed on uninstall.

The property is currently only supported by the CPack WIX Generator.

이전 항목

CPACK_NEVER_OVERWRITE

다음 항목

CPACK_START_MENU_SHORTCUTS
