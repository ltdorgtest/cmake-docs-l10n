INTERFACE_CXX_MODULE_SETS¶

Добавлено в версии 3.25.

Примечание

Experimental. Gated by CMAKE_EXPERIMENTAL_CXX_MODULE_CMAKE_API

Read-only list of the target's PUBLIC C++ module sets (i.e. all file sets with the type CXX_MODULES). Files listed in these C++ module sets can be installed with install(TARGETS) and exported with install(EXPORT) and export().

C++ module sets may be defined using the target_sources() command FILE_SET option with type CXX_MODULES.

See also CXX_MODULE_SETS.
