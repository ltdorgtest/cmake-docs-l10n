CMAKE_UNITY_BUILD_RELOCATABLE¶

Добавлено в версии 4.0.

This variable is used to initialize the UNITY_BUILD_RELOCATABLE property of targets when they are created. Setting it to true causes sources generated for CMAKE_UNITY_BUILD to #include the original source files using relative paths where possible.

Предыдущий раздел

CMAKE_UNITY_BUILD_BATCH_SIZE

Следующий раздел
