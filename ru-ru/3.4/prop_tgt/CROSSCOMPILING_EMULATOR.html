CROSSCOMPILING_EMULATOR¶

Use the given emulator to run executables created when crosscompiling. This command will be added as a prefix to add_test() test commands for built target system executables.

Предыдущий раздел

<CONFIG>_POSTFIX

Следующий раздел

CXX_EXTENSIONS
