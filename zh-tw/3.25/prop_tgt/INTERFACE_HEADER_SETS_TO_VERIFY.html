INTERFACE_HEADER_SETS_TO_VERIFY¶

在 3.24 版被加入.

Used to specify which PUBLIC and INTERFACE header sets of a target should be verified.

This property contains a semicolon-separated list of header sets which should be verified if VERIFY_INTERFACE_HEADER_SETS is set to TRUE. If the list is empty, all PUBLIC and INTERFACE header sets are verified. (If the project does not want to verify any header sets on the target, simply set VERIFY_INTERFACE_HEADER_SETS to FALSE.)
