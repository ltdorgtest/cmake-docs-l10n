CMAKE_PKG_CONFIG_DISABLE_UNINSTALLED¶

在 4.0 版被加入.

Enable / Disable the default "uninstalled" search behavior of the cmake_pkg_config() command. When this variable is false, package files with an "-uninstalled" suffix have higher priority than exact package name matches.
