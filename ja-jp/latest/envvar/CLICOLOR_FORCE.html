CLICOLOR_FORCE¶

Added in version 3.5.

This is a CMake Environment Variable. Its initial value is taken from the calling process environment.

Set to a non-empty value, other than 0, to tell command-line tools to print color messages even if not connected to a terminal. This is a common convention among command-line tools in general.

See also the CLICOLOR environment variable. CLICOLOR_FORCE, if activated, takes precedence over CLICOLOR.

See the CMAKE_COLOR_DIAGNOSTICS variable to control color in a generated build system.
