EXPORT_FIND_PACKAGE_NAME¶

注釈

Experimental. Gated by CMAKE_EXPERIMENTAL_EXPORT_PACKAGE_DEPENDENCIES.

Control the package name associated with a dependency target when exporting a find_dependency() call in install(EXPORT) or export(EXPORT). This can be used to assign a package name to a package that is built by CMake and exported, or to override the package in the find_package() call that created the target.

This property is initialized by CMAKE_EXPORT_FIND_PACKAGE_NAME.
