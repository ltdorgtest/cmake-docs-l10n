AUTORCC_SOURCE_GROUP¶

Added in version 3.9.

Name of the source_group() for AUTORCC generated files.

When set this is used instead of AUTOGEN_SOURCE_GROUP for files generated by AUTORCC.

AUTOMOC_TARGETS_FOLDER
