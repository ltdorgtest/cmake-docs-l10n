CMAKE_COMPILER_IS_GNUCXX¶

True if the C++ (CXX) compiler is GNU. Use CMAKE_CXX_COMPILER_ID instead.

上一个主题

CMAKE_COMPILER_IS_GNUCC

下一个主题

CMAKE_COMPILER_IS_GNUG77
